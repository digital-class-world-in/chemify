# **App Name**: Institute Dashboard

## Core Features:

- Dashboard Overview: Display key metrics such as total inquiries, admissions, and website visitors.
- Plan & Expiry: Show the current plan and its expiry date.
- Website Management Links: Provide quick links to website management and articles.
- Upcoming Birthdays: List upcoming student birthdays.
- Staff Management: Provide access to Staff Management related options.
- Financial Overview: Provide Financial analysis.

## Style Guidelines:

- Primary color: Light purple (#A382F0) to match the provided design's calming aesthetic.
- Background color: Very light gray (#F5F7FA) to maintain a clean and modern look.
- Accent color: A slightly darker shade of purple (#7952B3), analogous to the primary color, to highlight key elements.
- Body and headline font: 'PT Sans', sans-serif, combining a modern look and a little warmth.
- Use flat, vector-based icons in a style consistent with the provided design. The icons should be related to the data or actions they represent, using the accent color for highlights.
- The layout should be card-based, with rounded corners and consistent spacing to create a visually appealing and organized dashboard.
- Use subtle animations, such as fade-in effects or transitions, to enhance the user experience without being distracting.
